在软件开发领域，版本控制是非常重要的一环。它能够让我们追踪代码的变化、合作开发、管理不同的版本以及回滚到之前的状态。Git 是一个流行的分布式版本控制系统，本文将向你介绍如何使用 Git 来管理你的代码（以keil工程为例）。

在对工程操作之前，我们还需要对工程进行一项配置，创建.gitignore文件，在里面添加不需要追踪的文件，它们将不会被 Git 跟踪和提交，忽略他们，可以节省上传速度和空间。
